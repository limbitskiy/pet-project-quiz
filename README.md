# A small quiz app

**Server:**

```
- MongoDB
```

**Client:**

```
- Vue
- Nuxt
- Pinia
```

## How to start:

- clone repo
- `run docker-compose -f quiz-app.yml up -d`
- go to `localhost:3000`
